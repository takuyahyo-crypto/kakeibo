# ふたり家計簿 PWA — セットアップ手順（Google スプレッドシート版）

## 全体の流れ（Googleアカウントだけで完結）
```
スプレッドシート作成 → Apps Script を設定 → URL をコピー → アプリに貼り付け → 公開
```

---

## ステップ1: Googleスプレッドシートを作成

1. [Google スプレッドシート](https://sheets.google.com) を開く
2. 「新しいスプレッドシートを作成」
3. 名前を「家計簿データ」などに変更（任意）

---

## ステップ2: Apps Script を設定

1. スプレッドシートのメニューから **「拡張機能」→「Apps Script」** を選択
2. 左側に `コード.gs` が開く
3. 中身を**すべて削除**して、`code.gs` ファイルの内容を**まるごとコピーして貼り付け**
4. 💾 保存ボタン（Ctrl+S / Cmd+S）を押す

---

## ステップ3: ウェブアプリとして公開

1. 右上の **「デプロイ」→「新しいデプロイ」** をクリック
2. 左の歯車アイコン → **「ウェブアプリ」** を選択
3. 以下を設定：
   - 説明：`家計簿API`（任意）
   - **次のユーザーとして実行：「自分」**
   - **アクセスできるユーザー：「全員」**
4. 「デプロイ」→ Googleアカウントでの許可を求められたら **「アクセスを許可」**
5. 表示された **「ウェブアプリのURL」をコピー**

---

## ステップ4: URLをアプリに貼り付け

`config.js` を開いて、コピーしたURLを貼り付け：

```javascript
window.SCRIPT_URL = 'https://script.google.com/macros/s/xxxxxx/exec';
//                   ↑ ここにコピーしたURLを貼り付け
```

---

## ステップ5: Firebase Hosting でアプリを公開

### 5-1. Firebase CLI をインストール（初回のみ）
```bash
npm install -g firebase-tools
```

### 5-2. ログイン
```bash
firebase login
```

### 5-3. 初期化
```bash
cd ~/Desktop/Claude/家計簿PWA
firebase init hosting
```
質問への回答：
- `Create a new project` → プロジェクト名を入力（例: `kakeibo-app`）
- `public directory?` → `.`（ドット1つ）
- `single-page app?` → `N`
- `automatic builds with GitHub?` → `N`
- `index.html already exists. Overwrite?` → `N`

### 5-4. デプロイ
```bash
firebase deploy --only hosting
```

完了後に表示されるURL（例: `https://kakeibo-app.web.app`）がアプリのURLです。

---

## ステップ6: iPhoneにインストール

1. iPhoneの **Safari** でアプリのURLを開く
2. 下部の「共有」ボタン（四角に矢印のアイコン）をタップ
3. **「ホーム画面に追加」** → 「追加」
4. ホーム画面にアイコンが現れる ✓

---

## ステップ7: パートナーと共有

1. アプリを開く → **「新しい世帯を作成」** → 6文字のコードが生成される
2. 右上の👥アイコン → **「コードをコピー」** してパートナーに送る
3. パートナーはアプリを開いて **「参加する」** → コードを入力

## よくある質問

**Q: データはどこに保存される？**
A: Googleスプレッドシートに保存されます。スプレッドシートを開くと取引データが見られます。

**Q: リアルタイムで同期される？**
A: 10秒ごとに自動更新されます。追加・削除した場合は即時反映されます。

**Q: 費用はかかる？**
A: Googleアカウントとスプレッドシートは無料です。Firebase Hostingも無料枠内で使えます。

**Q: Apps Script URLの設定なしで使える？**
A: はい。`config.js` を設定しなければ、そのスマホのローカル保存モードで動作します（2人での同期は不可）。
